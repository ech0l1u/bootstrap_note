引用

本来bootsrap里引用没啥特点的，该 <blockquote>的该 <q>该咋样就咋样，但是 <footer>配合 <blockquote>就有点意思了，在bootstrap.css里： 

blockquote footer,
blockquote small,
blockquote .small {
 	display: block;
 	font-size: 80%;
 	line-height: 1.42857143;
	color: #777;
}
可以看到不止 <footer>，还有 <small>， .small都可以配合 <blockquote>这么玩。

列表

无样式列表

用 .list-unstyled，消除所有 list-style，不过这个类只针对 子元素，意思就是说子元素 <li里再嵌套的话，不会继承。 就像下面这样，如果想要继承效果的话，在嵌套里的 <ul> <ol> <dl>里添加类，不要在 <li>里加，会出问题的，，

  1. li----
  2. li---
  3. li--
    1. li----
    2. li---

内联列表

用 .list-inline，就是 display : inline-block再加一点 padding，同样的也只针对子元素，注意事项和上面是一样的

  1. li----
  2. li---
  3. li--
    1. li----
    2. li---
